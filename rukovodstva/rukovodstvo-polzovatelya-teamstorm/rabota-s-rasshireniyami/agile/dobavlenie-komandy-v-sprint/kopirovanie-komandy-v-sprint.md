# Копирование команды в спринт

1. Перейдите в спринт, в который требуется добавить команду.
2. Нажмите **Добавить команду** в правой верхней части экрана.
3. В открывшемся модальном окне нажмите **Скопировать из спринта**.
4. Выберите из списка спринт, из которого требуется скопировать команду.
5. При необходимости отредактируйте в отобразившихся полях количество дней отпуска каждого участника и количество часов, которые участник работает в день в рамках спринта.
6. Нажмите **Сохранить**.

{% hint style="info" %}
При копировании команды из другого спринта по умолчанию копируется количество часов, которые каждый участник работает в день. Количество дней отпуска не копируется.
{% endhint %}
