# Работа с разделом «Страницы»

Раздел «Страницы» — это интегрированная в TeamStorm система управления знаниями для распределенных команд.&#x20;

Раздел основан на принципах Wiki и существует на уровне рабочего пространства.&#x20;

Для перехода в раздел «Страницы»:

1. Перейдите в соответствующее пространство.
2. В панели навигации слева нажмите **Страницы**.

Если в разделе уже существуют документы, на основном экране будет отображена главная страница раздела. Слева будет отображена панель навигации по разделу.&#x20;

Если в разделе еще нет документов, будет отображена надпись **Пространство не содержит страниц** и кнопка **Создать страницу**.
