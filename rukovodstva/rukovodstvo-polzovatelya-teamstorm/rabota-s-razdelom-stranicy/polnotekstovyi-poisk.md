# Полнотекстовый поиск

Для поиска содержимого по тексту страниц:

1. Введите искомый текст в поле ввода в верхней части экрана.&#x20;
2. Выберите **Страницы**.

Результаты поиска будут отображены в виде списка под полем ввода.
