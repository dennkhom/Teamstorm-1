# Изменение типа задачи

1. Откройте карточку задачи, тип которой вы хотите изменить.
2. Нажмите кнопку <img src="../../../../.gitbook/assets/изображение (220).png" alt="" data-size="line"> в правом верхнем углу карточки и выберите **Изменить тип**.
3. В открывшемся модальном окне выберите из списка новый тип задачи.
4. Нажмите **Сохранить**.&#x20;

<figure><img src="../../../../.gitbook/assets/изображение (221).png" alt=""><figcaption></figcaption></figure>

Тип задачи будет изменен во всех представлениях.

В историю изменений задачи добавятся события:

* изменение типа (прошлое значение, новое значение);
* изменение статуса;
* запись об удалении атрибутов старого типа при условии их отсутствия в новом типе.
