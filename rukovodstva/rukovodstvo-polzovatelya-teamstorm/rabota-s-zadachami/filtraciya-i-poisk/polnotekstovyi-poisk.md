# Полнотекстовый поиск

Для поиска по текстам описаний задач:

1. Введите искомый текст в поле ввода в верхней части экрана.&#x20;
2. Выберите **Задачи**.

Результаты поиска будут отображены в виде раскрывающегося списка под полем ввода.
